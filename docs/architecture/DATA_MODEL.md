# Codex Data Model

Core entities:

- Candidate
- SourcePool
- LexiconEntry
- LanguagePack
- SearchMode
- Run
- Export
- Plugin
- Score
- ProvenanceRecord
- ReviewRecord
- Workspace
- Project

Every candidate must be traceable to source pool, raw input, normalised input, language pack, lexicon source, transformations, search mode, scoring model, engine version, and timestamp.
