# Codex Studio v0.1-alpha

A usable local interface powered directly by Codex Engine.

## Included
- source-pool editor
- language selection
- exact-anagram search
- ranked candidates
- provenance and score details
- local candidate persistence
- responsive interface
- health and history endpoints

## Launch
```bash
./scripts/run_codex_studio.command
```

Then open `http://127.0.0.1:8765`. Processing remains local.
